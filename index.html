One quick setup step

This copy of ExpenseIQ isn't fully connected yet. Open index.html and fill in the CONFIG object near the top of the script with your deployed Apps Script Web App URL, the shared secret token, and your Google OAuth Client ID (for Sign in with Google).

const CONFIG = {
  API_URL: 'https://script.google.com/macros/s/XXXX/exec',
  API_TOKEN: 'your-secret-token',
  GOOGLE_CLIENT_ID: 'xxxx.apps.googleusercontent.com'
};

Full step-by-step instructions are in README.md from the download package. Once all three values are set, the script is deployed as a Web App, and your GitHub Pages URL is registered as an authorized origin in Google Cloud Console, reload this page.

ℹ About ExpenseIQ

All data is stored securely in your Google Spreadsheet.
Sheet layout:
• Detailed_List — Expense records; headers at C2:I2, data from C3
• Ledger — Unified income & expense entries (Month-Year, Type, Title, Details, Amount)
• Fixed_Expenses — Recurring monthly store-type costs
• Budget — Monthly budget targets
• Settings — Item types, Unit types, Tracked items
• Allowed_Users — Gmail addresses permitted to sign in
• Sessions — Active sign-ins (delete a row here to force that device to sign out)
